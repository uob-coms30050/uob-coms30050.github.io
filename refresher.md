# Python Refresher 

The (Jean Golding Institute (JGI))[https://www.bristol.ac.uk/golding/] at the University of Bristol provides excellent training 
material and resources on data science and engineering. If your Python knowledge is rusty, 
go through the beginner and intermediate resources below. Look at the advanced material 
later in the teaching term.

- Beginner

-- Introduction to Python (part 1) https://bristol-training.github.io/intro-python-1/
-- Introduction to Python (part 2) https://bristol-training.github.io/intro-python-2/

- Intermediate

-- Introduction to Data Analysis in Python (Part 1)
https://bristol-training.github.io/data-analysis-python-1/

-- Introduction to Data Analysis in Python (Part 2)
https://bristol-training.github.io/data-analysis-python-2/

-- Introduction to NumPy
https://milliams.com/courses/intro_numpy/

- Advanced

-- Parralel Python: https://milliams.com/courses/parallel_python/
-- Accelerating Python: https://chryswoods.com/accelerating_python/


Look at JGI's Data Science and Research Software Engineering Training web page for the 
complete list of trainings available: https://bristol-training.github.io/courses.html

Other useful training material: 
- Software Engineering, Git and Version Control: https://bristol-training.github.io/courses.html#software-engineering
- High Performance Computing: https://bristol-training.github.io/courses.html#high-performance-computing-hpc
